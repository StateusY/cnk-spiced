# cnk-spiced
A small Crop and Kettle addon, with a few more recipes and ingredients.

HEY! This is an addon datapack! It will do nothing or break if Crop and Kettle ([Github Page](https://github.com/maybejake/crop-and-kettle) [Modrinth Page](https://modrinth.com/datapack/crop-and-kettle)), is not installed. Go install it!

Currently:
- Jambalaya recipes implemented
- Tiramasu planned and textured
- Cocktail planned and partially textured
- Dish 4 incomplete
- Dish 5 incomplete
- Lettuce Line textures and partially implemented
- Crops unimplemented
- Advancements partially implemented
- Item Script almost implemented (finish implementing generics and duplicates in recipes and custom data)
